Estamos buscando un desarrollador(a) blockchain altamente experimentado para construir un ecosistema blockchain completo. 
Este proyecto abarca el desarrollo de múltiples smart contracts y un frontend web interactivo, con miras a una futura 
integración con un juego Web3. El candidato ideal tendrá experiencia probada en Solidity, Web3 y proyectos que involucren 
NFTs y sistemas de staking. Los componentes clave del proyecto incluyen: Smart contract de NFTs: Desarrollo de un contrato 
inteligente para tokens no fungibles (NFTs) que incluirá la gestión de atributos únicos y la definición de rarezas. 
Smart contract de sistema de staking para NFTs: Implementación de un contrato inteligente que permita a los usuarios 
hacer staking con sus NFTs para obtener recompensas. Smart contract de token ERC-20: Creación de un token ERC-20 que 
servirá como mecanismo de recompensa dentro del ecosistema. Frontend web: Desarrollo de una interfaz de usuario intuitiva 
y robusta para permitir a los usuarios realizar operaciones de mint de NFTs, participar en el sistema de staking y gestionar 
sus tokens ERC-20. Integración futura con juego Web3: El diseño y la arquitectura deben considerar la preparación para una 
integración fluida con un futuro juego basado en tecnología Web3. Buscamos a alguien con un profundo conocimiento de la tecnología 
blockchain, capaz de escribir código seguro y optimizado, y con la habilidad de trabajar de forma independiente para entregar 
un producto de alta calidad.
